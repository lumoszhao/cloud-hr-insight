组织架构调整 AI算力 华为云 📅 2026-03-10

👥 具体人员变动

  • 张平安: 从华为云CEO → 华为云董事长(负责战略方向)
  • 周跃峰: 从华为数据存储产品线总裁 → 华为云CEO(负责执行,主抓ICT协同)
  • 变动原因: 强化战略+执行的协同,周跃峰有深厚ICT背景(存储),适合打通计算/存储/云服务

🏢 具体部门调整

  • 2025年8月(第一阶段): 撤销合并云BU下层组织,聚焦产品部/公有云服务部/研发部
  • 2025年11月(第二阶段): 研发体系整体并入ICT基础设施业务体系
  • 新成立5大研发产品线:
    • 基础设施云服务
    • Data&AI云服务
    • 数据库云服务
    • 安全云服务
    • HCS(Huawei Cloud Stack)

🎯 具体业务调整

  • "3+2+1"业务布局:
    • 3大基础业务:通算、智算、存储
    • 2大平台业务:AIPaaS、数据库
    • 1大核心业务:安全
  • 定位转变: 从"服务提供者" → AI时代"算力黑土地"和"卖水人"
  • 协同目标: 软硬芯协同(昇腾芯片+鲲鹏服务器+云服务)

💼 对招聘HR的影响

  • 销售岗位:
    • 重点招聘AI算力销售(智算业务是核心)
    • 需要懂ICT硬件(昇腾芯片、鲲鹏服务器)的销售人才
    • 政企云销售:政务云、金融云、制造业云仍是重点
  • 解决方案架构师:
    • 需要懂ICT基础设施(计算、存储、网络)的架构师
    • Data&AI云服务架构师成为热门岗位
    • 需要能讲清楚"软硬协同"解决方案的架构师
  • 招聘策略:
    • 从华为ICT产品线(存储、计算)挖人更容易
    • 重视有AI算力背景的候选人
    • 关注从竞争对手(阿里云、腾讯云)跳槽到华为的人才
