## 🚀 使用方式

### 方式一：自动更新（推荐）⭐
**无需任何操作**，GitHub Actions 会每天自动更新数据

### 方式二：手动更新
```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 运行更新脚本
python fetch_news.py
python scraper.py

# 3. 提交更改
git add .
git commit -m "chore: 手动更新数据"
git push
```

### 方式三：GitHub 手动触发
1. 进入 GitHub 仓库
2. 点击 "Actions" 标签
3. 选择 "Daily News Update"
4. 点击 "Run workflow"
