## 方法2：PowerShell 命令创建定时任务

复制以下命令到 PowerShell 执行：

```powershell
# 定义任务信息
$taskName = "云行业HR洞察-自动更新"
$scriptPath = "C:\Users\赵梓菡\WorkBuddy\Claw\netlify-deploy\schedule_update.bat"
$workDir = "C:\Users\赵梓菡\WorkBuddy\Claw\netlify-deploy"
$time = "18:00"

# 创建触发器（每天18:00）
$trigger = New-ScheduledTaskTrigger -Daily -At $time

# 创建操作（运行批处理脚本）
$action = New-ScheduledTaskAction `
    -Execute $scriptPath `
    -WorkingDirectory $workDir

# 设置任务参数
$settings = New-ScheduledTaskSettingsSet `
    -AllowStartIfOnBatteries `
    -DontStopIfGoingOnBatteries `
    -StartWhenAvailable

# 创建任务
Register-ScheduledTask `
    -TaskName $taskName `
    -Trigger $trigger `
    -Action $action `
    -Settings $settings `
    -Description "云行业HR洞察 - 每天自动抓取新闻并部署"

Write-Host "✅ 定时任务创建成功！" -ForegroundColor Green
```

## 常见问题

### Q1: 定时任务没有执行？

**解决方法：**
1. 检查任务计划程序中的任务状态
2. 查看历史记录是否有错误
3. 确保电脑在指定时间是开机状态
4. 检查 `schedule_update.bat` 路径是否正确

### Q2: Git 推送失败？

**解决方法：**
1. 确保已经配置好 Git 用户名和邮箱
2. 确保 Git 已授权访问 GitHub
3. 检查网络连接
4. 查看日志文件 `auto_update.log` 的错误信息

### Q3: 新闻抓取失败？

**解决方法：**
1. 检查网络连接
2. 更新 `auto_update.py` 中的新闻源 URL 和选择器
3. 查看日志文件了解具体错误
4. 可能需要更新 User-Agent 或添加代理

### Q4: 如何修改定时时间？

**解决方法：**
1. 打开任务计划程序
2. 右键任务 → **"属性"**
3. 切换到 **"触发器"** 标签
4. 双击现有触发器，修改时间
5. 点击 **"确定"** 保存

## 临时禁用/启用定时任务

### 禁用任务
```powershell

```

### 启用任务
```powershell
Enable-ScheduledTask -TaskName "云行业HR洞察-自动更新"
```

```powershell

```

## 自定义配置

### 修改抓取频率

编辑 `schedule_update.bat`，可以添加多个执行时间：

```batch
@echo off
REM 每天早上9点和下午6点执行

:loop
echo [%date% %time%] 开始自动更新...
python auto_update.py

REM 等待9小时后再执行
timeout /t 32400 /nobreak > nul
goto loop
```

### 添加更多新闻源

编辑 `auto_update.py`，在 `NEWS_SOURCES` 列表中添加：

```python
{
    'name': '新新闻源',
    'url': 'https://example.com/news',
    'selector': '.news-item',
    'title_selector': '.title',
    'date_selector': '.date',
    'link_selector': 'a'
}
```
